Air France et KLM appartiennent au même groupe et font régulièrement des **vols en code-share** : vous achetez un billet "Air France AF8000" mais l'avion est en réalité opéré par KLM. Ou inversement. Si le vol est retardé, qui vous doit l'indemnité ?

## Règle CE 261 : c'est la compagnie **opératrice** qui paie

L'article 2(b) du règlement CE 261/2004 est clair : l'obligation d'indemniser pèse sur **la compagnie qui opère effectivement le vol** ("operating carrier"), pas celle qui a vendu le billet ("marketing carrier").

**Exemple concret** :
- Vous achetez un billet Air France CDG → Lagos
- Numéro de vol : AF878
- Mais l'avion est en réalité un KLM B777 (vol code-share)
- Retard de 5h
- → **KLM doit payer les 600€**, pas Air France

## Comment identifier l'opérateur réel

Plusieurs méthodes :

- **Sur la carte d'embarquement** : la compagnie qui apparaît dessus est généralement l'opérateur
- **Le numéro du vol** : un préfixe AF avec un n° à 4 chiffres + opéré par KL (mention "opéré par") = KLM
- **Flightradar24** : recherche du n° de vol → montre le code opérateur réel
- **L'application Air France/KLM** : précise "opéré par"

## Les cas Air France ↔ KLM les plus fréquents

- Code AF, opéré KL : vols vers Afrique de l'Est (Nairobi, Entebbe), via Amsterdam
- Code KL, opéré AF : vols Asie via Paris (Tokyo, Shanghai)
- Code AF, opéré HV (Transavia) : vols vacances Méditerranée, et certaines routes Afrique

## Pourquoi la compagnie marketing vous dirige vers l'autre

Air France et KLM ont des **systèmes de réclamation distincts**, même s'ils sont du même groupe. Quand vous écrivez à Air France pour un vol opéré KLM, votre réclamation est souvent **rejetée pour "compagnie incompétente"** — perte de temps.

Solution : adressez **directement** votre réclamation à la compagnie **opératrice** (KLM dans l'exemple).

## Code-share avec d'autres compagnies

Le principe est le même pour tous les codes-share :

- **Air France ↔ Delta** : Delta opérateur sur les vols transatlantiques = recours direct à Delta
- **Air France ↔ China Eastern** : compagnie non-UE → si départ hors UE, pas de CE 261
- **KLM ↔ Kenya Airways** : Kenya Airways non-UE → CE 261 uniquement vol départ AMS/CDG
