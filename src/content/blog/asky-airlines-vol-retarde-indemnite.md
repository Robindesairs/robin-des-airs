ASKY Airlines est une compagnie panafricaine basée à Lomé (LFW, Togo), opérée en partenariat avec Ethiopian Airlines. Elle dessert plus de 20 destinations en Afrique de l'Ouest et Centrale depuis son hub de Lomé. Pour les voyageurs européens, ASKY est souvent la connexion finale après un vol Bruxelles ou Paris vers Lomé, puis vers leur ville de destination en Afrique.

## ASKY = compagnie non-UE : règle clé

ASKY Airlines est enregistrée au Togo — compagnie non-européenne. Le CE261/2004 s'applique donc uniquement si :

- Le vol **départ d'un aéroport UE** (Paris, Bruxelles, Amsterdam…)
- OU la compagnie est UE (ce qui n'est pas le cas d'ASKY)

| Trajet | Éligible CE261 ? |
|---|---|
| Paris/Bruxelles → Lomé (ASKY) | ✅ Oui — départ UE |
| Lomé → ville africaine (ASKY, intra-Afrique) | ❌ Non |
| Ville africaine → Lomé → Paris (retour ASKY) | ❌ Non (départ hors-UE) |

## Comment ASKY s'insère dans les voyages Europe-Afrique

Le trajet typique pour un passager basé en Europe :

1. **Paris → Bruxelles → Lomé** (Brussels Airlines, UE) → couvert
2. **Lomé → [Cotonou / Douala / Abidjan / Bamako…]** (ASKY, intra-Afrique) → sur billet séparé = non couvert

Si les deux tronçons sont sur **un seul billet** avec un code de réservation unique : c'est la distance totale Paris → destination finale qui compte → **600€** probable.

## Destinations ASKY depuis Lomé

ASKY dessert notamment : Abidjan, Bamako, Cotonou, Dakar, Douala, Kinshasa, Lagos, Libreville, Niamey, Ouagadougou, Accra, Bangui, Yaoundé, Conakry, Freetown.

Ces destinations sont exactement les corridors de la diaspora francophone en France.

## Conseil pratique

Pour avoir la couverture CE261 sur les vols retour depuis l'Afrique, choisissez Brussels Airlines (via Bruxelles) ou KLM (via Amsterdam) pour le tronçon Afrique-Europe — ces compagnies UE couvrent les deux sens.
