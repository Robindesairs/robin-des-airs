**Jusqu'à 600 € par passager** restent à réclamer sur un vol charter retardé de 3 h ou plus, même réservé via un tour-opérateur. On vérifie gratuitement vos droits, 0 € d'avance.

Un vol charter est un vol affrété par un tour-opérateur (TUI, FRAM, Marmara, Look Voyages, Nouvelles Frontières, etc.) pour une destination spécifique, généralement en package vacances. Quand il est retardé, vos droits **se cumulent** : ceux du CE 261 et ceux du Code du tourisme.

## 1er recours — Le CE 261/2004 contre la compagnie aérienne

Tout vol charter au départ d'un aéroport de l'UE est couvert par le CE 261, comme un vol régulier. Si retard ≥ 3h à l'arrivée ou annulation, vous avez droit à :

- 250€ si distance ≤ 1 500 km
- 400€ si 1 500 – 3 500 km
- **600€ par passager** si > 3 500 km (DOM-TOM, Afrique, Asie)

Cible : **la compagnie qui opère le charter**. Ex : pour un TUI fly Paris-Dakar, c'est TUI fly Belgium ou autre, pas l'agence TUI.

## 2e recours — Code du tourisme contre le tour-opérateur

Si vous avez acheté un **package "voyage organisé"** (vol + hôtel + transfert), vous bénéficiez aussi du **Code du tourisme** (art. L. 211-15) :

- **Réacheminement** si annulation
- **Remboursement intégral** si vous renoncez au voyage suite à une modification majeure
- **Indemnité pour perte de jouissance** : 1, 2, 3 jours de vacances perdus → remboursement proratisé du séjour

C'est l'**agence ou le tour-opérateur** qui est votre interlocuteur — pas la compagnie aérienne.

## Les deux peuvent se cumuler

**Exemple** :
- Vous achetez chez TUI un package Paris-Punta Cana 8 jours
- Vol charter aller retardé de 6h → vous perdez la première journée
- → Indemnité **CE 261 = 600€/passager** (vol > 3 500 km) versée par la compagnie
- → Indemnité **Code du tourisme = environ 1/8 du prix du séjour** versée par TUI pour la journée perdue

## Les pièges à éviter

- **"Bon d'achat" du tour-opérateur** : ne signez rien comme "règlement définitif" — vous perdriez vos droits CE 261
- **"Force majeure"** invoquée par le tour-opérateur : l'argument vaut pour lui, mais **pas pour la compagnie** au sens CE 261 (sauf circonstance extraordinaire prouvée)
- **Délai de prescription** : 2 ans pour le Code du tourisme, **5 ans** pour le CE 261 en France (art. 2224 du Code civil)
