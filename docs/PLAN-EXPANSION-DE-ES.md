# Plan d'expansion blog DE / ES — 23 articles prioritaires par langue

> **Objectif** : passer de 7 → 30 articles dans chaque langue (DE & ES).
> **Stratégie** : prioriser les routes diaspora + compagnies locales + sujets à fort volume de recherche dans chaque marché.

## Méthode de priorisation

1. **Routes** : couvrir les axes diaspora majeurs (DACH ↔ Afrique pour DE, Espagne ↔ Amérique latine + Maghreb pour ES).
2. **Compagnies** : couvrir la compagnie nationale + les low-cost dominants.
3. **Procédure** : adapter les articles "que faire si...", "combien je touche..." au cadre légal local (Allemagne : § 195 BGB → 3 ans de prescription ; Espagne : 5 ans + ley AVCC).
4. **Saisonnier** : adapter aux pics de voyage locaux (Oktoberfest, Karneval pour DE ; Semana Santa, agosto pour ES).

## Workflow de traduction recommandé

Pour ne pas perdre 6 mois en traduction manuelle :

1. **Préparer un prompt système Claude** (ou Mistral / DeepL Pro) avec :
   - Le glossaire FR ↔ DE ↔ EN ↔ ES des termes juridiques aériens
   - Les contraintes : conserver JSON-LD, conserver `<style>`, conserver structure HTML
   - Adapter le cadre légal local (BGB, AESA, etc.)

2. **Workflow batch** :
   ```bash
   # 1. Copier l'article FR source
   cp blog/vol-retarde-X-Y-indemnite.html de/blog/verspaeteter-flug-X-Y-entschaedigung.html

   # 2. Lancer une traduction LLM avec contexte du glossaire
   #    (script Python utilisant l'API Anthropic / OpenAI)

   # 3. Validation humaine (15 min / article)

   # 4. Ajouter aux sitemaps DE / ES
   ```

3. **Coût estimé** :
   - Claude Sonnet API : ~$0.04 / article (3-4 K tokens × ~$3 / M tokens)
   - 46 articles = **$2** API + 12 h de relecture humaine
   - Beaucoup plus rentable qu'un traducteur freelance (~50 €/article)

4. **Cadence proposée** :
   - **Semaine 1** : 8 articles routes DE + 8 articles routes ES (les 16 plus rentables)
   - **Semaine 2** : 6 compagnies DE + 6 compagnies ES
   - **Semaine 3** : 5 procédure DE + 5 procédure ES
   - **Semaine 4** : 4 spécifiques DE + 4 spécifiques ES + ajout aux sitemaps
