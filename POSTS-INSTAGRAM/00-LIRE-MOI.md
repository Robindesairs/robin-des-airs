## 📲 Comment les mettre sur ton téléphone (le plus simple)

**Option A — AirDrop (le plus rapide) :**
1. Ouvre ce dossier dans le Finder.
2. Sélectionne les 4 images (clic + Cmd pour tout prendre).
3. Clic droit → **Partager → AirDrop** → ton iPhone.
4. Sur le tél : Instagram → ➕ → **Story** → choisis l'image → **Ta story**. (En Story, pas besoin de légende.)

**Option B — WhatsApp Web (pour les avoir dans WhatsApp) :**
1. Ouvre **web.whatsapp.com**.
2. Ouvre une discussion (la tienne / « Moi » ou un contact).
3. Clique le trombone 📎 → **Photos et vidéos** → sélectionne les 4 images → **Envoyer**.
4. Récupère-les sur ton tél depuis WhatsApp, puis poste.
